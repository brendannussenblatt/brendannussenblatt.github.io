Spider-Man History Website

A multi-page site that organizes Spider-Man’s evolution by decade, creators, characters, and key moments.

Overview

I built this site to present Spider-Man’s history in a structured, visual format—more like an archive than a fan page. The goal was to make the character’s growth easy to explore while keeping navigation clear and content easy to scan.

View the live website →

What I Did

  • Designed a multi-page information architecture (decades, creators, villains, etc.)
  • Built consistent navigation and layout patterns across pages
  • Focused on readability, content hierarchy, and visual styling that supports scanning

Tools

HTML CSS JavaScript Responsive Layout

Outcome

This project strengthened my ability to structure content-heavy experiences and design layouts that feel organized, consistent, and easy to navigate.

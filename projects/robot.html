Arduino Tank Robot

A hardware + software build using Arduino and servos to create a tank-style robot with scripted movement.

Overview

I built a small tank robot (nicknamed “Turny”) that uses servo motors for movement and an Arduino for control logic. The goal was to create reliable, repeatable motion sequences and test basic interaction patterns.

Project Demo

▶️ Watch the robot fully operational (YouTube Short)

What I Did

  • Wired and configured servo motors and controls (including button-triggered sequences)
  • Wrote Arduino code to run consistent movement patterns and timing
  • Iterated on debugging and reliability (inputs, delays, and movement tuning)

Tools

Arduino C/C++ Servo Control Hardware Prototyping
